How to Join the Server

Hey! Want to join our Minecraft server? Follow these steps:

1. Open Powershell

Open the windows search and type "powershell".

2. Run the following command

irm mcinstall.narwal.llc | iex

3. Join the Server

Follow the instructions in powershell. The server will automaticaly populate in your server list once complete.

Note:

If the modpack isn't working, such as after an update, re-run the above command to repair the installation. This will automatically back up any of your config files, and can be run freely.
