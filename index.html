Join the Server

Get set up in minutes. Follow the steps below to install the modpack and connect.

Step 01

Open PowerShell

Open the Windows search bar and type powershell and open it.

Step 02

Run the Install Command

Copy and paste the command below into your PowerShell window and press Enter.

irm mcinstall.narwal.llc | iex
Step 03

Join the Server

Follow the automated instructions in PowerShell. Once complete, the server should automatically appear in your Minecraft server list!

If the server doesn't appear: Manually add it using the server IP: mc.narwal.llc

💡 Repairing your install: If the modpack stops working or an update is released, simply re-run the command above. It will safely back up your configuration files and repair the installation.
