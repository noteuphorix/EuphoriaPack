Privacy Policy

Effective date: June 12, 2026  ·  Applies to all Narwal LLC game servers

No marketing use No analytics tracking Never sold or shared Only what the server needs
Overview

The short version

We collect only what is necessary to run our game servers. That means things like your character data, preferences, and playtime — the information the server software stores by default so that your progress is saved between sessions. Nothing beyond that is collected, and what is collected is never used for marketing, advertising, analytics, or sold to any third party.

What we collect

Data collected by default

Our servers run on Space Station 14 and Minecraft server software. These platforms store a standard set of data as part of normal operation. We do not extend or supplement this with any additional data collection.

Data Why it's stored Collected
Character data Saves your in-game character, appearance, and job preferences between sessions Yes
Playtime records Tracks time played per role, used for job unlock requirements Yes
Account identifiers Links your in-game session to your authenticated account Yes
Ban and admin records Standard server moderation logs Yes
Marketing or ad data — Never
Analytics or tracking — Never
Data sold to third parties — Never
How it's used

Server operations only

Any data stored is used exclusively to make the server function — saving your character, enforcing bans, and keeping your progress intact. We do not use it for anything else. We do not run analytics pipelines, build user profiles, or share data with advertisers or data brokers.

Third parties

No selling, no sharing

Your data is not sold, licensed, or shared with any third party. The only external services involved are the underlying game platforms (Space Station 14 authentication and Minecraft account authentication) which operate under their own respective privacy policies.

Changes

Policy updates

If this policy ever changes, the effective date at the top of this page will be updated. We'll also announce any meaningful changes in our Discord. Continuing to use the server after a change takes effect constitutes acceptance of the updated policy.

Contact

Questions?

If you have any questions about how your data is handled, the best place to reach us is our Discord server. We're happy to answer anything.
